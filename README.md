## 💡 The Business Value (The Prototype-First Approach)
One of the biggest bottlenecks in Data Analytics and BI development is stakeholder alignment. Building complex data pipelines and DAX measures takes time, and delivering a product that doesn't meet the client's exact vision leads to endless revisions.

This project is a **Rapid Interactive Mockup** built to solve that exact problem. 
By presenting this fully functional frontend prototype to decision-makers first, we can:
1. Validate KPIs and visual layout instantly.
2. Gather accurate feedback on the spot.
3. Save weeks of backend development time.
4. Ensure the final Power BI / Python deployment is 100% aligned with business needs.

## ✨ Key Features
* **Functional Filters:** Interactive Date and Segment dropdowns.
* **Synthetic Data Engine:** A custom JavaScript engine that generates realistic, dynamic data based on user filter selections.
* **Animated Visualizations:** Interactive Sales Funnel and Churn Trend charts that update in real-time.
* **Corporate UI/UX:** Clean, minimalist, and responsive design mimicking high-end SaaS analytics platforms.

## 🛠️ Tech Stack
This lightweight prototype was built without heavy backend frameworks to ensure rapid deployment:
* **HTML5** for structure.
* **Tailwind CSS** (via CDN) for rapid, responsive styling.
* **Chart.js** (via CDN) for dynamic data visualization.
* **Vanilla JavaScript** for the synthetic data generation and interactivity.

## ⚙️ How to Run Locally
Since this is a single-file prototype with no external dependencies or database connections, running it is incredibly simple:
1. Clone this repository: `git clone https://github.com/yourusername/ecommerce-crm-mockup.git`
2. Open the `index.html` file directly in any modern web browser.
3. Interact with the filters to see the Synthetic Data Engine in action!
